## Installation

### Requirements

- Laravel `8` or higher
- PHP `7.4` or higher

### Composer

```bash
composer require burning-cloud-system/laravel-entity
```

## Basic Usage

### Entity class creation

```bash
php artisan make:entity [table_name]
```
